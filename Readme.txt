CNavigating は、ComiketCatalogBrowser で出力された複数のCSVファイルを統合します。


●使い方
統合するサークルリストを保存します。version 1.0は保存形式「Windows版V2 ComicMarket83」に対応しています。
CNavigatingで「開く」をクリックしサークルリストを開きます。
統合するサークルリストをすべて開いたら「保存」でリスト上のサークルリストを統合して保存します。
「設定」を開くと色のカスタム情報を設定することができます。


●統合の仕様
各サークルリストに登録されているサークルがすべて色番号、コメントを保持して統合されます。
複数のサークルリストで競合が起きたときは、コメントを連結して保存し、色番号は色がついていて番号の小さい方を優先して保存します。
過去のサークル情報などは保存されません。


●動作環境
.NET Framework 3.0ランタイムがインストールされているWindows PC。Vista以降なら多分大丈夫。
